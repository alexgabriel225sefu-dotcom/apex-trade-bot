# Apex Trade Bot 🤖

AI-powered crypto trading bot. Deploy with one click on Railway.

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new/template?template=https://github.com/alexgabriel225sefu-dotcom/apex-trade-bot)

## Setup
After deploying, add these variables in Railway → Variables:

| Variable | Value |
|----------|-------|
| `LICENSE_KEY` | Your key from purchase email |
| `EXCHANGE` | `binance` |
| `BINANCE_API_KEY` | Your Binance API key |
| `BINANCE_API_SECRET` | Your Binance API secret |
| `GROQ_API_KEY` | Free key from console.groq.com |
| `PAPER_TRADING` | `true` (start safe) |
| `PAPER_BALANCE` | `100` |
